# Front-end

Utilizando Vite 4.4.9 + React 18.2.0 + NodeJs 18.15.0

## Actualizar NodeJS 
```bash
sudo npm cache clean -f
sudo npm install -g n
sudo n stable
```

## Ejecucion:
Ubicandose en la carpeta de la repo:
Instalar las dependencias
```bash
$ npm install
```

Iniciar el servidor local
```bash
$ npm run dev --port:5173
```
